# Как запустить

1) Качаем архив, распаковываем в удобное место
2) Открываем в Visual Studio Code
3) Открываем консоль, и пишем `npm install`, ждем
4) Вводим `npm run dev`
5) Открываем в браузере http://localhost:5173/
